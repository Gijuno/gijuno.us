1. 인간, 사회, 환경에 대한 통합적 관점 이해
  1. 세상을 바라보는 다양한 관점
    - 시간적 관점 : 현상이나 사건의 현 모습이 있기까지의 배경과 맥락
      (보통 과거의 자료를 바탕으로 함)
    - 공간적 관점 : 위치, 장소, 분포 양상, 형성과정, 이동과 네트워크
      (인간, 사회, 환경의 상호관계)
    - 사회적 관점  사회 구조 및 제도의 측면
       (정치적, 경제적, 문화적 제도의 영향 / 시민의 권리 의무)
    - 윤리적 관점 : 도덕적 차원의 기준
      (사회의 일을 도덕적 기준으로 판단)

  2. 통합적 관점
    - 의미 : 사회 현상을 통합적으로 살펴봄 -> 각각의 관점이 아닌 종합적으로
    - 필요성 : 복잡한 현상은 한가지 관점으로 해결 불가 (한계가 있음), 근본적 해결책 찾기 가능


2. 행복의 의미와 기준
  1. 삶의 목적
    - 궁극적 목적이라기보단 "다른 무언가를 이루기 위한 수단"

  2. 시대와 지억에 따른 행복의 기준
    - 일반적 의미 : 삶의 충분한 만족감, 기쁨
    - 공통된 기준 : 의식주 기본 욕구 충족, 신체,정서적 건강, 좋은 인간관계
    - 시대적 기준
      - 헬레니즘 : 안녕히계세요 여러분
      - 중세 : 신의 선택 -> 천국
      - 근대 이후 : 자유, 평등
    - 지역적 기준
      - 빈곤 : 기본적 의식주 충족, 무질병
      - 풍요 : 인간관계
      - 무민주 : 민주

  3. 진정한 행복의 실현
    - 객관적 기준 : 주거, 소득, 고용, 수명, 교육수준
    - 주관적 기준 : 삶의 만족도, 일상 소확행
    - 진정한 행복 실현 : 삶의 질 전체 향상, 객관적/주관적 기준 모두 충족


3. 행복한 삶 실현 조건
  1. 질 높은 정주 환경
    - 필요성 : 낙후된 환경 -> 인간다운 삶 X
    - 필요 요소
      - 깨끗한 자연환경 : 물, 대기, 토양
      - 쾌적한 사회환경 : 치안, 교육, 문화 서비스
    - 국가의 노력 : 도심 녹지 확보, 문화/의료 시설 확층

  2. 경제적 안정
    - 필요성 : 기본적 생계 유지, 필요 충족
    - 요소 : 일정 수준 이상 소독 지속
    - 국가의 노력 : 경제 활성화, 일자리 확층, 실업급여 제공, 사회 보험 마련
  
  3. 민주주의 발전
    - 필요성 : 시민 인권 존중, 자신이 원하는 삶 추구 가능
    - 요소 : 민주적 제도, 시민 정치 참여 -> 적극적 투표권, 정책 결정 과정의 시민 참여

  4. 도덕적 실천
    - 필요성 : 도덕적 성찰 -> 더 나은 사람, 의미/행복 느낌
    - 노력 : 보편적 가치에 따른 행동, 그른일 바로잡기, 더불어 살아가기


4. 자연환경과 인간 생활
  1. 기후가 생활에 미치는 영향
    1. 기후와 기후 요인
      - 기후 : 일정한 지억에서 장기간 걸쳐 나타나는 대기 상태
      - 기후 요인 : 기온, 강수량, 바람 같은 기후 요소가 지역차에 영향을 주는 요인 (위도 ,해발고도, 수륙분포)
    2. 세계의 기후 지역
      - 열대, 건조, 온대, 냉대, 한대 (저위도 -> 고위도)
      - 기후 특성 따라 "생활, 산업 달라짐" 
      - 열대 : 앏고 편한 옷, 이동식 경작, 개방형 가옥(고상가옥)
      - 건조 : 온몸 감싸는 헐렁한 옷, 유목/오아시스 농업, 흙/이동식 가옥
      - 온대 : "인간 생활에 유리, 인구 밀도 높" 계절에 따라 옷 달라짐, 농경에 유리 (농축산물 생산)
      - 냉대 : "인간 생활에 불리" 밀/잡곡 경작, 폐쇄적 가옥(통나무집)
      - 한대 : 겨울 길고 추워 거주 힘듬, 수렵생활/순록 유목, 가죽 옷/가옥 발달
  
  2. 지형이 생활에 미치는 영향
    - 미치는 영향 : 거주 공간과 생활 양식에 영향 미침
    - 산지 : 해발 고도 높고, 급경사 많음 -> 인간 생활에 불리 / 임업,가축 사육, 밭농사, 관광 사업 발달
    - 평야 : 지표면 평탄 -> 인간 생활, 농경에 유리 / 도시발달, 경제활동
    - 해안 : 육지와 바다 모두 이용 가능 -> 수산업, 관광 산업 발달, 항구,산업단지 조성


5. 안전하고 쾌적한 환경에서 살 권리
  1. 자연의 삶 위협
    1. 자연재해 : 인간을 위협하는 자연현상
      - 자연으로부터의 위협 방지 힘듬
      - 기후 변화로 이상기후 발생, 횟수 규모 증가
      - 우리나라는 홍수, 태풍, 폭설등 기후적 요인 피해 주로 발생 / 최근엔 지진
    2. 기후적 요인에 의한 자연재해
      - 홍수 : 일시에 많은 비 -> 시가지, 농경지 침수
      - 가뭄 : 오랫동안 적은 비 -> 식수, 농업용수 부족
      - 폭설 : 일시에 많은 눈 -> 교통 혼란, 시설물 붕괴
      - 폭염 : 일정 기준 이상 기온 상승 -> 인명 및 재산 피해
      - 열대 저기압(태풍) : 강풍과 호우 동반, 홍수/풍랑
    3. 지형적 요인
      - 지진 : 땅이 갈라짐 -> 건축물, 도로 붕괴 / 단시간 다피해
      - 화산 : 용암, 화산재 분출 -> 농작물, 주거지 피해 / 항공 운항 지장
      - 산사태 : 호우, 지진 -> 토사가 가옥, 도로 덮침

  2. 안전하고 쾌적한 환경에서 살 권리 보장
    1. 안전하고 쾌적환 환경에서 살 권리
      - 인간의 존엄성 보장 위한 기본권
      - 헌법에 시민, 환경권 관련 조항 명시
    2. 시민의 권리 보장 위한 국가의 역할
      - 내진설계, 조기 경보 -> 피해 최소화 위한 사전 대비
      - 자연재해 발생시 즉각 복구 위한 시스템 마련
      - 재해민 보상, 지원 대책 마련
    3. 스스로 권리 보장 위한 시민의 노력
      - 재해 대비 안전교육 참여
      - 재해 발생시 행동 요령 따라 대응


6. 인간과 자연의 바람직한 관계
  1. 자연을 바라보는 인간의 관점
    1. 인간의 관점
      - 자연환경은 인간에게 의식주의 원천 제공
      - 자연재해 통해 인간에게 위협
      - 자연과 인간의 관계에 따라 삶의 방향 달라짐 -> 인간 중심주의 or 생태 중심주의
    2. 인간 중심주의
      - 의미 : 인간 최고 인간이 탑이다
      - 특징 : 자연은 인간의 삶을 위한 도구
      - 한계 : 자원 고갈, 환경 오염 -> 이간에게 큰 피해
    3. 생태 중심주의
      - 의미 : 인간의 이익보다 자연 전체의 균형 안전
      - 특징 : 인간 포함, 자연 전체가 하나다
      - 한계 : 생태 중심주의 지나친 강조는 모든 자연 개발 중단 야기

  2. 인간과 자연의 관계
      - 유기적 관계 : 인간의 삶과 자연은 유기적으로 연결
      - 공존 관계 : 인간과 자연은 서로 대립하지 않고 공존
      - 공존의식 : 환경 문제 해결 위해 미래세대까지의 생존과 생태계 전체 보전 의식 필요


7. 환경 문제의 발생과 해결 위한 노력
  1. 오늘날 세계의 환경 문제
    1. 지구 온난화 : 가장 큰 문제
      - 원인 : 화석 연료 사용 증가, 산림 파괴, 무분별 목초지/농경지 확대
      - 영향 : 이상 기후, 극지방/고산 빙하 축소, 해수면 상승
    2. 사막화
      - 원인 : 장기간 가뭄, 과도한 방목/개간, 람림 벌채
      - 영향 : 사막 주변지역 토양 황폐화, 식량 생산량 감소, 기근, 황사
    3. 산성비
      - 원인 : 공장, 자동차 등 황산화물, 질소산화물 -> 대기오염
      - 영향 : 삼림 파괴, 호수/토양 산성화, 구조물/건물 부식
    4. 열대림 파괴
      - 원인 : 부분별 벌목, 농경지 확대
      - 영향 : 지구 자정 능력, 생물 종 다양성 감소
    5. 우리나라의 환경 문제
      - 원인 : 산업화로 인한 인구,자원 소비 증가
      - 영향 : 하천,해양 오염, 미세먼지,황사,스모그(대기오염 심화)
  
  2. 환경 문제 해결 위한 노력
    1. 정부의 노력
      - 자연 생태계 보고, 자원 소비 절감, 오염 물질 배출 규제
      - 환경 정책, 에너지 절약, 방안 홍보
    2. 시민 단체의 노력
      - 환경 문제 시민운동, 전파
      - 정부의 정책, 기업의 활동 감시,비판
    3. 기업의 노력
      - 환경 오염 물질 배출 감소
      - 자원 재활용, 에너지 효율성, 신재생 에너지 확대 기술 개발

1. 인간, 사회, 환경과 행복
 1 - 인간, 사회, 환경에 대한 통합적 관점 이해
  1) 시간적 관점 : 어떤 현상이나 사건의 현재 모습이 있기까지의 시대적 배경과 맥락
  2) 공간적 관점 : 다양한 현상을 위치와 장소, 분포 양상과 형성 과정, 이동과 네트워크 등의 공간적 맥락
  3) 사회적 관점 : 현상이 나타나게 된 배경을 사회 구조 및 제도의 측면에서 분석하고 대안을 살펴보는 것
  4) 윤리적 관점 : 인간의 행위가 도덕적 차원에서 인정받기 위한 기준을 탐색하고 바람직한 삶의 모습을 살펴봄.
              다양한 사회 현상을 도덕적 가치에 따라 평가하고 사회가 나아가야 할 규번적 방향을 성정하는데 도움.
  5) **통합적 관점** : 시간적, 공간적, 사회적, 윤리적 관점을 함게 고려.
         인간, 사회, 국가, 지구 공동체 및 환경을 개별 학문의 경계를 넘어 종합적으로 이해.
         복잡한 현상을 정확히 이해할 수 있고, 문제에 대한 근본적인 해결책 찾기 가능.
 2 - 행복의 의미와 기준
  1) 행복이란 삶에서 충분한 만족감이나 기쁨을 느끼는 상태.
      의식주에 대한 기본적 욕구 충족과 신체적, 정서적 건강, 친밀한 인간 관계 등 필수적인 요소.
  2) 객관적 기준 : 주거, 소득, 고용, 수명 등 행복 실현의 중요한 기준
  3) 주관적 기준 : 삶의 만족도나 일상생활에서 느기는 행복감 등.
  4) **통합** : 최근의 행복 관련 지수들은 객관적 기준과 더불어 개인의 주관적 만족감까지 통합적으로 고려하여 행복을 측정.
 3 - 행복한 삶을 실현하기 위한 조건
  1) 질 높은 정주 환경 : 낙후된 환경에서는 인간다운 삶 X, 안락한 보금자리 필요. 생존, 기본적인 삶, 지속가능한 환경, 유대감 등.
            이를 위해서 자연환경 보호. 물, 대기, 토양 보호. 치안 서비스, 보건 및 위생 서비스, 학교 및 교육 서비스 등 필요.
  2) 경제적 안정 : 경제적 안정이 실현되어야 기본적인 생계 유지와 자아실현의 기회를 가질 수 있음. (필요성)
              일정 수준 이상의 소득이 보장, 일자리를 확충, 실업 급여 제공, 사회 보험 마련등 다양한 복지 정책 필요. (조건)
  3) 민주주의의 발전 : 민주 국가일수록 시민의 인권이 존중되며, 시민이 원하는 삶의 방식 자유롭게 추구 가능.
  4) 도덕적 실천 : 바람직한 삶에 대한 성찰 (삶의 성장), 삶의 의미와 행복 느낌 (구성원 모두의 행복) (필요성)
              인권이나 정의와 같은 보편적 가치, 도덕적 성찰, 역지사지 (조건)

2. 자연환경과 인간
 1 - 자연환경 : 의식주를 비롯한 인간 생활에 커다란 영향을 미쳐 세계의 지역마다 주민들의 생활 모습이 다르게 나타난다.
  1) 기후 : 위도, 해발 고도, 수륙 분포 등 다양한 요인에 의해 달라진다.
   (1) 열대 기후 : 적도 주변에 분포하고 일사량이 많아 연중 기온 높다. 집의 구조는 단순하고 개방적이며 창문을 크게 만듦. 옷은 천으로 만든 간편한 옷
   (2) 건조 기후 : 강수량이 적어서 물이 부족해 식물 성장과 거주에 불리. 주민들은 유목, 오아시스에서 농업. 옷은 온몸을 감싼 옷. 집은 흙집이나 이동식.
   (3) 냉대 기후 : 고위도에 분포하므로 겨울이 길고 추워 인간 생활에 불리. 집 창문이 작고 폐쇄적인 구조. 난방 시설 잘 갖춤.
   (4) ***온대 기후*** : 중위도에 분포하므로 계절의 변화가 뚜렷하고 기온이 온화함. 계절별로 입는 옷 달라짐. 각종 농축산물 생산. 다양한 요리 발달.
   (5) 한대 기후 : 북극과 남극 주변에 분포하므로 겨울이 길고 추워 인간이 거주 힘듦. 주민들은 수렵이나 순록 유목.

   (2) 평야 : 해발 고도가 낮고 땅이 평평해서 농사에 유리, 생활에 적합. 사람들이 모여살음. 최근에는 도시가 발달, 산업 시설 들어서는등 경제의 주요.
   (3) 해안 : 육지와 바다가 맞닿아 있어 두 곳을 모두 이용 가능. 어업이나 양식업에 종사. 평야가 있는 곳은 농업. 최근에는 해안 지형을 이용한 관광 산업,

   (3) 폭설 : 한꺼번에 많은 양의 눈이 내림. 교통 혼란을 초래하고 약한 구조물 붕괴.

   (2) 화산 활동 : 용압과 함께 가스, 화산재 등이 분출하면서 농작물과 주거지 등을 덮침, 화재 발생, 화산재는 바람을 타고 이동하여 다른 지역에 피해.
